## 🚀 Featured Open Source Projects

### 🛡️ [MythrilUI](https://github.com/TheK1ER/MythrilUI)
A custom, hardware-accelerated 2D UI framework built entirely from scratch in C++ using OpenGL 3.3.
* Features a custom **9-Slice scaling algorithm** for distortion-free rectangular UI components.
* Advanced text rendering using **FreeType** with dynamic word-wrapping and smart ellipsis (`...`) clipping.
* Polymorphic render pipeline utilizing a modern, type-safe variant batching system.

### 📐 [LiteMatrix-CPP](https://github.com/TheK1ER/LiteMatrix-CPP)
A lightweight, standalone C++ matrix library for linear algebra and machine learning operations. Features dot products, slicing, concatenation, and Xavier initialization.

### 🧠 [DAG-NN-CPP](https://github.com/TheK1ER/DAG-NN-CPP)
A lightweight, zero-dependency Neural Network framework built from scratch in C++. Features a Directed Acyclic Graph (DAG) architecture for custom branching and merging operations.

### 📄 [C++ CSV Reader](https://github.com/TheK1ER/CSVReader)
A lightweight and flexible C++ CSV parsing library. Parse specific rows, columns, or ranges into memory with safe type conversion and efficient memory management.
