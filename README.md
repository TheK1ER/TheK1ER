# Hi, I'm TheK1ER! 👋

I am a **Systems & Graphics Engineer** specializing in low-level C++ development, hardware-accelerated computer graphics, and machine learning architectures built completely from scratch.

I strongly believe in a **zero-dependency mindset**. Instead of relying on bloated, black-box engines or high-level AI frameworks, I thrive on building core systems—from custom graphics renderers to high-performance dataset parsers and raw neural network mathematics—directly on top of the metal.

---

## 🚀 Featured Projects

### 🛡️ [MythrilUI](https://github.com/TheK1ER/MythrilUI)
A custom, hardware-accelerated 2D UI framework built entirely from scratch in C++ using OpenGL 3.3 (GLFW / GLAD).
* Features a custom **9-Slice scaling algorithm** for distortion-free rectangular UI components.
* Advanced text rendering using **FreeType** with dynamic word-wrapping and smart ellipsis (`...`) clipping.
* Polymorphic render pipeline utilizing a modern, type-safe variant batching system.

### 📐 [Matrix](https://github.com/TheK1ER/Matrix)
A lightweight, highly optimized C++ mathematical library designed to handle raw linear algebra, vector, and matrix transformations. This serves as the core computational backbone for both my graphics pipelines and deep learning models.

### 🧠 [NeuralNetwork](https://github.com/TheK1ER/NeuralNetwork)
A custom artificial neural network (ANN) engine built completely from scratch in C++. Developed to demonstrate the underlying mathematical principles of deep learning (such as backpropagation and gradient descent) utilizing raw memory management and math optimizations instead of high-level AI libraries.

### 📄 [readCSV](https://github.com/TheK1ER/readCSV)
A lightning-fast, lightweight CSV parser written in modern C++. Engineered to handle and stream massive datasets with minimal memory overhead, feeding raw training data directly into my custom neural network pipelines.
